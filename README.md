# COVID-19 Detection using CNN

This project detects COVID-19 from chest X-ray images using a Convolutional Neural Network (CNN) and PyQt5 GUI.

## Technologies Used
- Python
- TensorFlow / Keras
- PyQt5
- NumPy

## Features
- Train CNN model
- Classify chest X-ray images
- GUI application

## Dataset
Chest X-ray images of COVID and Normal cases.

## Run the Project
pip install tensorflow pyqt5 numpy pillow

python main.py
